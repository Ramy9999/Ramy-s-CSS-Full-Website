ML VideoToText and Topic Modelling

Machine Learning (ML), OCR, OpenCV, Pytesseract, ML LDA (Latent Dirichlet Allocation) Topic Modelling




Results:-





This is an ML project where you can upload any video containing text of choice and it will be transformed to actual text and that text will be analyzed for ML topic modelling and seeing which are the most frequent words in the video to label and classify the video's topics.

How to use and run the Google Colab Notebook:
1- Download the Google Colab Notebook via the google drive link below.
2- Open Google Colab online, sign in or create an account using your google account or gmail, and press File then Upload notebook, then choose and upload the downloaded Google Colab Notebook.
3- Press Runtime then press Run all; to run all the cells.
4- After the second cell finishes running, go to its output and press Restart Runtime.
5- Then press Runtime and press Run all, to run all cells again.
6- In the third cell, after it runs, it will ask for uploading a video file, upload any video of choice.
7- Then in the fifth cell, after it runs, it will ask to upload the same video again, upload that again.

And that is it, now the images of the video are saved and their text is written to a text file and then that same text file is analyzed and used for Machine Learning Topic Modelling, you can see the results in the rest of the cells in and after the section: Topic Modelling from fetched Text From Video

NASA InSight API

Python, Flask, Bootstrap, and Twilio!

This API gets you the most recent Mars InSight Images faster than searching on NASA's site! Get SMS updates when new images have been received from the API!

During deployment I learned about setting up a Flask environment with templates and static folders directories, along with understanding how Bootstrap can work with forms. Python works easily with other libraries such as WTForms, and Twilio. Setting up your environment variables can be done in a few ways and I chose to use the os module. Flask requires specific syntax in the HTML file such as 'url_for' and code blocks for specific HTML components like Header, Footer, and Script tags. My goal is to get the SMS and email up and running along with an SQLite database to store emails and phone numbers so I can send updates to users when new images are posted to the API. I have another few iterations planned out and am excited to get this out there to those who love learning about our planet Mars! Keep reaching for the planets!
